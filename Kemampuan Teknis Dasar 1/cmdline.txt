1. Listing files (ls) : melihat directories / file yang terdapat di komputer

Desktop  Downloads  Pictures  snap    test

..             .config    .mozilla  snap                       .vscode
Adi            Desktop    Music     .ssh
.bash_history  Documents  Pictures  .sudo_as_admin_successful
.bash_logout   Downloads  .pki      Templates
.bashrc        .gnupg     .profile  test
terlihat semua directory/ file yang tersembunyi ataupun yang tidak tersembunyi.

       5. Delete File (menghapus file)
       “rm filename”
       blk@blk-HP-Pavilion-Gaming-Laptop-15-dk1xxx:~$ ls
Adi      Documents  latihan  Pictures  snap       test   Videos
Desktop  Downloads  Music    Public    Templates  tugas

6. membuat directory baru
“mkdir directoryname”
=> blk@blk-HP-Pavilion-Gaming-Laptop-15-dk1xxx:~$ mkdir latihan
7. Renaming Directory (mv)
“blk@blk-HP-Pavilion-Gaming-Laptop-15-dk1xxx:~$ mv latihan latihan1”
8. Removing Directories (rmdir)
“blk@blk-HP-Pavilion-Gaming-Laptop-15-dk1xxx:~$ rmdir latihan1”
9. The History Command (history)

“blk@blk-HP-Pavilion-Gaming-Laptop-15-dk1xxx:~$ history”
